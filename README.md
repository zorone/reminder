# How to get start

## CLI basics
Change directory
'''
cd path/to/dest
'''
_You can use 'TAB' and 'shift+TAB' to quickly select location_

Change directory with different Drive
'''
X:
'''

Show directory
'''
dir
'''

Pull files from git
'''
git pull
'''

Add files to git
'''
git add -A
'''

Add commit
'''
git commit -a
'''

Exit nano
'''
Ctrl+X -> Enter
'''

Push to commit
'''
git push
'''
